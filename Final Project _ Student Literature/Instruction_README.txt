1. This is a NetBeans project. So the project should be opened using NetBeans.
2. Absolute Layout must be added to the library.(Right click on Libraries>Add Library>(select)Absolute Layout).
3. mysql-connector jar file should be added to the librery.
4. There are two string variables in the MainClass.java file named 'user' and 'pass'. Currently user = root, pass = tiger
	Change this two variables according to your MySql username and password.
5. Import the database file using the MySql workbench. name of the schema is dtb.
6. Run the program.
